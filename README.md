# Computational Logic of Time (CLT)

CLT is a minimal, system-independent framework for evaluating temporal propositions over discrete state sequences.

It corresponds to Linear Temporal Logic (LTL) interpreted over a single linear trace, while eliminating transition systems and branching semantics.
